# 키워드 광고 대행 클라이언트 요청 관리 시스템 사용 가이드

키워드 광고 운영 대행 업무 수행 시 다수의 클라이언트 담당자로부터 발생하는 문의 및 요청 사항을 체계적으로 접수·추적·처리하기 위한 웹 기반 관리 시스템입니다.

## 1. 시스템 접속

### 1.1 로그인

1. 웹 브라우저에서 시스템 URL에 접속합니다.
2. 로그인 페이지에서 **이메일**과 **비밀번호**를 입력합니다.
3. **로그인** 버튼을 클릭합니다.

### 1.2 로그인 후 기본 화면

- **클라이언트 담당자**: 요청 현황 페이지(`/client/requests`)로 이동
- **광고 운영 담당자/관리자**: 대시보드 홈 페이지(`/`)로 이동

### 1.3 로그아웃

- 화면 좌측 사이드바 하단의 **로그아웃** 버튼을 클릭합니다.

## 2. 사용자 역할

| 역할 | 설명 | 접근 가능 메뉴 |
|------|------|---------------|
| **클라이언트 담당자** | 광고주 측 담당자로서 요청을 등록하고 진행 상황을 확인합니다. | 요청 등록, 요청 현황 |
| **광고 운영 담당자** | 클라이언트의 요청을 접수하고 처리합니다. | 대시보드, 요청 처리 |
| **관리자** | 시스템 전체를 관리하고 통계를 확인합니다. | 대시보드, 요청 처리, 통계, 시스템 관리 |

## 3. 클라이언트 담당자 가이드

클라이언트 담당자는 광고 운영에 관련된 요청을 등록하고 처리 현황을 확인할 수 있습니다.

### 3.1 요청 등록

**메뉴 위치**: 사이드바 > 요청 등록

#### 요청 등록 절차

1. **요청 등록** 메뉴를 클릭합니다.
2. 요청 정보를 입력합니다:
   - **제목**: 요청 내용을 요약한 제목 (필수)
   - **요청 유형**: 예산 변경, 키워드 추가/삭제, 광고 소재 수정 등 (필수)
   - **광고 플랫폼**: 네이버, 카카오, 구글 등 (필수)
   - **긴급도**: 일반, 긴급, 최우선 (필수)
   - **희망 완료일**: 요청 처리 희망 날짜
   - **상세 내용**: 요청에 대한 자세한 설명 (필수)
3. **파일 첨부** (선택사항):
   - 파일 선택 버튼을 클릭하여 관련 파일을 첨부합니다.
   - 여러 파일을 첨부할 수 있습니다.
4. **등록** 버튼을 클릭하여 요청을 제출합니다.

#### 요청 유형 설명

| 유형 | 설명 |
|------|------|
| 예산 변경 | 광고 예산 증액, 감액, 재배분 요청 |
| 키워드 추가/삭제 | 광고 키워드 추가 또는 삭제 요청 |
| 광고 소재 수정 | 광고 문구, 이미지, 링크 등 수정 요청 |
| 타겟팅 변경 | 광고 노출 대상, 지역, 시간대 변경 요청 |
| 리포트 요청 | 광고 성과 리포트 요청 |
| 계정 설정 | 광고 계정 관련 설정 변경 요청 |
| 기타 | 위 유형에 해당하지 않는 요청 |

### 3.2 요청 현황 조회

**메뉴 위치**: 사이드바 > 요청 현황

#### 요청 목록 확인

1. 본인이 등록한 모든 요청 목록을 확인할 수 있습니다.
2. **검색**: 요청번호, 제목으로 검색할 수 있습니다.
3. **필터**: 상태(전체, 접수대기, 처리중, 완료, 보류)별로 필터링할 수 있습니다.

#### 요청 상세 보기

1. 목록에서 **보기** 아이콘(눈 모양)을 클릭합니다.
2. 상세 정보 팝업에서 다음 내용을 확인할 수 있습니다:
   - 요청 기본 정보 (제목, 유형, 플랫폼, 긴급도, 상태)
   - 상세 내용
   - 첨부 파일 (클릭하여 다운로드 가능)
   - 담당자 코멘트 및 첨부 파일

#### 요청 수정

- **접수대기** 상태인 요청만 수정할 수 있습니다.
- 상세 보기 팝업에서 **수정** 버튼을 클릭합니다.
- 내용 수정 후 **저장** 버튼을 클릭합니다.
- 첨부 파일 추가/삭제도 가능합니다.

### 3.3 담당자 코멘트 확인

요청 상세 보기 화면 하단에서 담당자가 작성한 코멘트를 확인할 수 있습니다.

- 파란색 배경의 코멘트 영역에서 담당자의 답변을 확인합니다.
- 담당자가 첨부한 파일도 다운로드할 수 있습니다.
- 코멘트 작성 일시와 작성자 정보가 표시됩니다.

## 4. 광고 운영 담당자 가이드

광고 운영 담당자는 클라이언트의 요청을 접수하고 처리합니다.

### 4.1 대시보드

**메뉴 위치**: 사이드바 > 대시보드 (홈)

대시보드에서 다음 정보를 한눈에 확인할 수 있습니다:

#### 통계 카드
- **오늘 접수**: 오늘 새로 접수된 요청 수
- **미처리 요청**: 접수대기 상태인 요청 수
- **처리중**: 현재 처리 중인 요청 수
- **기한 임박**: 희망 완료일이 가까운 요청 수

#### 주요 섹션
- **긴급도별 미처리 요청**: 긴급도에 따른 미처리 요청 현황
- **내 담당 요청**: 본인에게 배정된 요청 목록
- **처리 기한 임박**: 희망 완료일이 임박한 요청 알림
- **오늘 접수된 요청**: 오늘 새로 등록된 요청 목록

### 4.2 요청 처리

**메뉴 위치**: 사이드바 > 요청 처리

#### 요청 목록 확인

1. 모든 클라이언트의 요청 목록을 확인할 수 있습니다.
2. **상태 탭**: 전체, 접수대기, 처리중, 완료, 보류 탭으로 빠르게 필터링
3. **검색**: 요청번호, 제목, 클라이언트명으로 검색
4. **긴급도 필터**: 전체, 최우선, 긴급, 일반으로 필터링

#### 요청 상세 처리

1. 목록에서 제목을 클릭하거나 **보기** 아이콘을 클릭합니다.
2. 상세 페이지에서 요청 내용과 첨부 파일을 확인합니다.

#### 상태 변경

1. 상세 페이지 상단의 **상태** 셀렉트박스를 클릭합니다.
2. 변경할 상태를 선택합니다:
   - **접수대기**: 아직 확인하지 않은 요청
   - **처리중**: 현재 작업 중인 요청
   - **완료**: 처리가 완료된 요청
   - **보류**: 추가 확인이 필요하여 보류된 요청
3. 상태가 즉시 변경됩니다.

#### 담당자 배정

- 미배정 요청의 경우 담당자를 배정할 수 있습니다.
- 요청 목록에서 **담당자 배정** 아이콘(사람+ 모양)을 클릭합니다.

### 4.3 코멘트 작성

요청 상세 페이지 하단에서 코멘트를 작성할 수 있습니다.

#### 코멘트 작성 절차

1. 코멘트 입력란에 내용을 작성합니다.
2. **내부용 코멘트** 체크박스:
   - 체크 시: 담당자/관리자만 볼 수 있는 내부 메모
   - 체크 해제 시: 클라이언트도 볼 수 있는 공개 코멘트
3. **파일 첨부** (선택사항):
   - 파일 선택 버튼을 클릭하여 파일을 첨부합니다.
   - 여러 파일을 첨부할 수 있습니다.
4. **코멘트 작성** 버튼을 클릭합니다.

#### 코멘트 수정

1. 본인이 작성한 코멘트의 **수정** 버튼을 클릭합니다.
2. 내용을 수정합니다.
3. 첨부 파일을 추가하거나 삭제할 수 있습니다.
4. **저장** 버튼을 클릭합니다.

#### 코멘트 삭제

1. 본인이 작성한 코멘트의 **삭제** 버튼을 클릭합니다.
2. 확인 메시지에서 **확인**을 클릭합니다.

## 5. 관리자 가이드

관리자는 운영 담당자의 모든 기능에 더해 통계 확인과 시스템 관리 기능을 사용할 수 있습니다.

### 5.1 통계

**메뉴 위치**: 사이드바 > 통계

- 요청 처리 현황에 대한 통계 리포트를 확인할 수 있습니다.
- 기간별, 유형별, 담당자별 통계를 조회할 수 있습니다.

### 5.2 시스템 관리

**메뉴 위치**: 사이드바 > 시스템 관리

#### 클라이언트 관리

클라이언트 담당자 계정을 관리합니다.

**클라이언트 추가**
1. **클라이언트 추가** 버튼을 클릭합니다.
2. 정보를 입력합니다:
   - 이메일 (로그인 ID로 사용)
   - 비밀번호
   - 이름
   - 회사/부서명
   - 연락처
3. **추가** 버튼을 클릭합니다.

**클라이언트 수정**
1. 목록에서 **수정** 버튼을 클릭합니다.
2. 정보를 수정 후 **저장** 버튼을 클릭합니다.

**비밀번호 변경**
1. 목록에서 **비밀번호 변경** 버튼을 클릭합니다.
2. 새 비밀번호를 입력 후 **변경** 버튼을 클릭합니다.

**클라이언트 삭제**
1. 목록에서 **삭제** 버튼을 클릭합니다.
2. 확인 메시지에서 **확인**을 클릭합니다.

#### 담당자 관리

광고 운영 담당자 계정을 관리합니다.

**담당자 추가**
1. **담당자 추가** 버튼을 클릭합니다.
2. 정보를 입력합니다:
   - 이메일 (로그인 ID로 사용)
   - 비밀번호
   - 이름
   - 역할 (operator 또는 admin)
3. **추가** 버튼을 클릭합니다.

**담당자 수정/삭제/비밀번호 변경**
- 클라이언트 관리와 동일한 방식으로 진행합니다.

## 6. 공통 기능

### 6.1 파일 첨부 및 다운로드

#### 파일 첨부
- 요청 등록/수정 시 파일을 첨부할 수 있습니다.
- 코멘트 작성 시에도 파일을 첨부할 수 있습니다.
- **파일 선택** 버튼을 클릭하여 파일을 선택합니다.
- 여러 파일을 선택할 수 있습니다.

#### 파일 다운로드
- 첨부된 파일명을 클릭하면 다운로드됩니다.
- 원본 파일명이 유지됩니다.

#### 파일 삭제
- 수정 모드에서 파일 옆의 **X** 버튼을 클릭하여 삭제합니다.

### 6.2 요청 상태 설명

| 상태 | 설명 | 색상 |
|------|------|------|
| 접수대기 | 요청이 등록되었으나 아직 담당자가 확인하지 않은 상태 | 노란색 |
| 처리중 | 담당자가 요청을 확인하고 처리 중인 상태 | 파란색 |
| 완료 | 요청 처리가 완료된 상태 | 초록색 |
| 보류 | 추가 확인이 필요하여 처리가 보류된 상태 | 회색 |

### 6.3 긴급도 설명

| 긴급도 | 설명 | 색상 |
|--------|------|------|
| 일반 | 일반적인 처리 우선순위 | 초록색 |
| 긴급 | 빠른 처리가 필요한 요청 | 주황색 |
| 최우선 | 가장 높은 우선순위로 즉시 처리 필요 | 빨간색 |

### 6.4 사이드바 네비게이션

화면 좌측의 사이드바를 통해 메뉴를 이동할 수 있습니다.

- 각 메뉴를 클릭하면 해당 페이지로 이동합니다.
- 현재 위치한 메뉴는 하이라이트 표시됩니다.
- 사이드바 하단에 로그인된 사용자 정보와 로그아웃 버튼이 있습니다.
